# UI/UX Flow

## User Journeys
1. **Onboarding**
   - Splash → Sign In/Sign Up → Verify → Home Feed.
2. **Create Cue**
   - Home Feed → New Cue → Compose → Publish → Home Feed.
3. **Engage**
   - Home Feed → Cue Detail → React/Comment → Back.
